Kebijakan Privasi

Terakhir diperbarui: 24 Juli 2026

Dokumen Kebijakan Privasi ini menjelaskan bagaimana aplikasi Tanjakan Maut: Offroad Adventure dikembangkan oleh Strata mengelola informasi pengguna.

Ringkasan

Aplikasi ini merupakan game offline. Kami tidak mengumpulkan, menyimpan, mengirim, maupun menjual data pribadi pengguna ke server mana pun. Seluruh data permainan disimpan secara lokal pada perangkat pengguna.

1. Informasi yang Kami Kumpulkan

Game ini tidak mengumpulkan informasi pribadi seperti:

  • Nama
  • Alamat Email
  • Nomor Telepon
  • Alamat Rumah
  • Lokasi GPS
  • Kontak
  • Foto atau Video
  • Data Keuangan
  • Informasi Identitas
  • Riwayat Penelusuran
  • Riwayat Pembelian

Kami juga tidak meminta izin akses terhadap data pribadi tersebut.

2. Data Permainan

Aplikasi hanya menyimpan data permainan secara lokal pada perangkat menggunakan mekanisme penyimpanan Flutter seperti SharedPreferences.

Contoh data yang disimpan:

  • Jumlah koin
  • Status upgrade kendaraan
  • Level yang telah dibuka
  • Pengaturan suara
  • Skor tertinggi
  • Progres permainan

Data tersebut tidak pernah dikirim ke server mana pun.

3. Penggunaan Data

Data lokal hanya digunakan untuk:

  • Menyimpan progres permainan.
  • Menyimpan pengaturan pengguna.
  • Menampilkan skor tertinggi.
  • Melanjutkan permainan.

4. Berbagi Data

Kami tidak membagikan data pengguna kepada pihak ketiga.

Aplikasi ini tidak menggunakan layanan yang mengumpulkan data pribadi seperti:

  • Firebase Analytics
  • Firebase Crashlytics
  • Google Analytics
  • Facebook SDK
  • OneSignal
  • AppsFlyer

5. Layanan Pihak Ketiga

Game ini dibuat menggunakan framework Flutter. Framework tersebut digunakan hanya sebagai teknologi pengembangan aplikasi dan tidak digunakan untuk mengumpulkan data pribadi pengguna.

Apabila pada versi mendatang aplikasi menggunakan layanan pihak ketiga seperti Google Play Games, Google AdMob, Firebase, atau layanan lainnya, Kebijakan Privasi ini akan diperbarui.

6. Keamanan Data

Karena seluruh data permainan disimpan secara lokal, keamanan data bergantung pada keamanan perangkat pengguna.

Menghapus data aplikasi atau menghapus instalasi aplikasi akan menghapus seluruh progres permainan secara permanen.

7. Privasi Anak

Aplikasi ini tidak secara khusus ditujukan kepada anak-anak di bawah usia 13 tahun. Kami tidak dengan sengaja mengumpulkan informasi pribadi dari anak-anak.

8. Hak Pengguna

Karena kami tidak mengumpulkan data pribadi pengguna, tidak terdapat data yang perlu diminta untuk diakses, diperbaiki, ataupun dihapus dari server kami.

Pengguna dapat menghapus seluruh data permainan kapan saja melalui menu Pengaturan Android dengan memilih "Hapus Data" atau menghapus aplikasi.

9. Perubahan Kebijakan Privasi

Kami dapat memperbarui Kebijakan Privasi ini sewaktu-waktu. Perubahan akan dipublikasikan melalui halaman ini beserta tanggal pembaruan terbaru.
