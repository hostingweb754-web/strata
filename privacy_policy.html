Kebijakan Privasi

Terakhir Diperbarui: 15 Juli 2026

Kebijakan Privasi ini menjelaskan bagaimana Strata selaku pengembang game Tanjakan Maut: Offroad Adventure ("Game") mengumpulkan, menggunakan, dan melindungi informasi Anda saat Anda menggunakan aplikasi seluler kami.

Informasi Utama: Game kami dirancang sebagai game offline yang menghargai privasi penggunanya. Kami tidak mengumpulkan data pribadi sensitif Anda ke server eksternal.

1. Pengumpulan Informasi

Kami membatasi pengumpulan data seminimal mungkin untuk memastikan kenyamanan bermain Anda:

  • Data Progres Game (Penyimpanan Lokal): Game ini menyimpan progres permainan Anda (jumlah koin yang dikumpulkan, status pembelian peningkatan mesin/ban/suspensi/stabilisasi, serta lintasan yang telah dibuka) secara lokal di perangkat Anda menggunakan penyimpanan terenkripsi/persistent (seperti SharedPreferences).
  • Data Non-Pribadi (Statistik Gameplay): Selama permainan berlangsung, Game hanya memproses data fisika (seperti kecepatan kendaraan, putaran RPM, sisa bahan bakar) secara lokal untuk mensimulasikan sistem fisika Forge2D dan tidak membagikannya ke luar perangkat Anda.

2. Data Pribadi Sensitif

Kami tidak pernah meminta, mengumpulkan, menyimpan, atau mentransmisikan informasi pribadi apa pun dari perangkat Anda, termasuk namun tidak terbatas pada:

  • Nama, alamat email, nomor telepon, atau identitas fisik lainnya.
  • Lokasi GPS presisi atau informasi jaringan nirkabel secara berkala.
  • Akses ke daftar kontak, riwayat panggilan, berkas media foto/video, atau data akun sosial lainnya pada perangkat Anda.

3. Penggunaan Informasi

Data progres lokal yang disimpan di perangkat Anda hanya digunakan untuk:

  • Memastikan progres peningkatan (upgrade) dan koin Anda tidak hilang saat Game ditutup atau dihentikan (pause).
  • Menghitung pencapaian jarak terjauh yang berhasil Anda capai di masing-masing stage (Gurun, Salju, Bukit Country).

4. Keamanan Data

Keamanan progres game Anda bergantung pada keamanan perangkat fisik Anda sendiri. Menghapus instalan aplikasi (uninstall) atau menghapus cache aplikasi melalui pengaturan sistem Android akan menghapus seluruh data koin dan progres upgrade Anda secara permanen, karena data tersebut tidak dicadangkan secara online di server kami.

5. Perubahan pada Kebijakan Privasi

Kami dapat memperbarui Kebijakan Privasi ini dari waktu ke waktu untuk menyesuaikan dengan fitur game yang baru atau regulasi hukum yang berlaku. Kami menyarankan Anda untuk meninjau halaman ini secara berkala untuk mengetahui perubahan apa pun. Perubahan berlaku segera setelah dipublikasikan di halaman ini.
